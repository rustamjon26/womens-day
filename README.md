# 8-Mart Delight 🌷

Premium Women's Day Celebration Portal. A festive, elegant, and interactive web experience designed to celebrate International Women's Day on March 8th.

## ✨ Features

- **🌸 Personalized Greetings**: Enter a name to generate a beautiful, personalized festive card.
- **✨ Compliment Generator**: A constant stream of warm and inspiring words for women.
- **🎁 Surprise Gift Box**: Interactive gift box experience with festive messages and confetti.
- **💝 Qualities Grid**: A curated collection of the beautiful traits that make women special.
- **🎵 Music Player**: Curated spring-themed background music for a festive mood.
- **🍂 Animated Atmosphere**: Floating flower petals and smooth, premium animations throughout.
- **📱 Fully Responsive**: Optimized for a premium experience on mobile, tablet, and desktop.
- **🇺🇿 Full Uzbek Localization**: Visible UI and content are 100% in the Uzbek language.

## 🛠 Tech Stack

- **React + Vite**: High-performance frontend foundation.
- **TypeScript**: Type-safe development.
- **Tailwind CSS**: Custom premium styling with glassmorphism.
- **Framer Motion**: Smooth, high-quality animations.
- **Lucide React**: Beautiful, consistent iconography.
- **Canvas Confetti**: Festive celebration effects.

## 🚀 Getting Started

1. **Install dependencies**:

   ```bash
   npm install
   ```

2. **Run development server**:

   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```
